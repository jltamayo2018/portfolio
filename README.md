# Portfolio Jose Luis Tamayo

Web estática lista para publicar en Netlify.

## Cómo desplegar
1. Crea un repositorio en GitHub.
2. Sube todos estos archivos.
3. Entra en Netlify > Add new site > Import an existing project.
4. Conecta GitHub y selecciona el repositorio.
5. Build command: vacío.
6. Publish directory: `.`
7. Deploy.

## Personalización rápida
- Cambia `images/profile-placeholder.svg` por tu foto real y mantén el mismo nombre.
- Cambia los enlaces de LinkedIn y GitHub en `index.html`.
- Añade tu PDF real en `pdfs/cv-jose-luis.pdf`.
